## Build Setup (client)

```bash
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn run dev

# build for production with minification
yarn run build
```

## Build Setup (Server)

```bash
# install dependencies
npm install

# serve with nodemon at localhost:8081
npm start
```
